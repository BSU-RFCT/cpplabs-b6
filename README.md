# cpplabs

Лабораторные работы по программированию на C++.

## Описание

Этот репозиторий содержит решения лабораторных работ по курсу программирования на C++. Каждая лабораторная работа представлена отдельным исполняемым файлом.

## Требования

- CMake версии 3.31 или выше
- Компилятор C++ с поддержкой стандарта C++20
  - GCC 10+ / Clang 10+ / MSVC 2019+

## Сборка проекта

### Linux / macOS

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

### Windows

```bash
mkdir build
cd build
cmake ..
cmake --build . --config Release
```

После сборки исполняемые файлы будут находиться в директории `build/bin/`.

## Запуск лабораторных работ

Каждая лабораторная работа — это отдельная программа. Запустить можно следующим образом:

```bash
# Linux / macOS
./build/bin/1_1

# Windows
.\build\bin\Release\1_1.exe
```
